# FizzBuzzRun

Command-line program to show FizzBuzz values from 1 to 24  

### install
```bash
go install github.com/neetsdkasu/fizzbuzzrun@latest
```

### run
```bash
%GOPATH%\bin\fizzbuzzrun.exe
```
